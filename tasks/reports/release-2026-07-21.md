## Known issues (ship-acceptable)

- **Antigravity**: weekly/5h quota not readable locally (in-app only); transcripts hold no
  token counts, so no real token total — plan + credits only. (`tasks/reports/provider-connections-2026-07-21.md`)
- **Gemini**: requires a one-time `gemini` CLI sign-in; otherwise an honest empty state.
- **Cline**: appears only when recent sessions exist on disk.
- **Menu-bar item**: agents couldn't screenshot it in a packed/notched bar — verify the
  status item + burn mark render on the target menu bar. Near-black bars dim the mark's ink
  region (silhouette-stroke fallback available if wanted).
- **opencode**: `immutable=1` read may lag the newest un-checkpointed WAL frames (negligible;
  the tradeoff that stops the open from failing).
- **Website** `providers`/tier fill values are illustrative placeholders, not live data.
